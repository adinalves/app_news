<h2 align="center">How to Use </h2>

   ```   
   - Clone this repository:
   $ git clone https://github.com/adinalves/app_news.git app_news

   - Enter in directory:
   $ cd app_news

   - For install dependencies:
   $ flutter pub get

   - Run the app: 
   $ flutter run
   ```
